Before modifying existing code:

1. Analyze the current implementation.
2. Explain what will change.
3. Explain what could break.
4. Preserve existing functionality.
5. Minimize changes whenever possible.
6. Do not rewrite working code without a clear reason.
7. Prefer extending existing code over replacing it.
8. If a refactor is needed, explain why.

Treat existing working code as valuable.
